# Welcome to the Astar algorithm visualizer

Made as an experiment to learn more about pathfinding and javascript.

# Usage

Open index.html in your browser (looks the most ideal in fullscreen)

# Key bindings

'r': Resets the pathfinder
'c': Clears the grid of all obstacles
'e': Places the destination at the mouse cursor
's': Places the start at the mouse cursor
